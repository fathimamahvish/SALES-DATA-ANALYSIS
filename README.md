# Sales Data Analysis Project

This project analyzes sales performance across different products, regions, and customer segments. The goal is to identify key revenue drivers, profitability trends, and actionable business insights that support data-driven decision-making.

| Product | Product name |
| Category | Product category |
| Quantity | Units sold |

---

## üîç Key Insights

1. **Top Products** generate a significant portion of total revenue. These items should be prioritized in marketing and stock planning.
2. **Sales vary across regions**, with some regions performing notably better. Low-performing regions may require targeted promotions.
3. **Customer segments** show different spending patterns, which can inform customized sales strategies.
4. **Profit margins** vary by product category, indicating opportunities to optimize pricing or cost structure.
